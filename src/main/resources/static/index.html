Sincronizar configuración

¿Qué hace la sincronización?
Envía la configuración actual (zonas, usuarios y parámetros de cada edificio) a todos los nodos FOG registrados. Si un nodo no responde en 2 segundos, queda marcado OFFLINE y la configuración queda pendiente de aplicar la próxima vez que esté disponible.
